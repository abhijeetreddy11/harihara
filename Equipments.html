This printer delivers print jobs fast, using healthier Latex water-based inks and produces durable, scratch-resistant prints comparable to hard-solvent and eco-solvent inks.

Environment-friendly latex platforms are used with Green Guard certified inks for indoor and outdoor signage.

Automatic laminator, laminates the print with high speed efficiency. This machine is capable of roll to roll lamination which can run in both modes hot and cold.
